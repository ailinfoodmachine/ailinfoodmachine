# 内容资料投放说明

你可以把真实产品资料按下面目录放入项目中，后续替换 `data/products.json` 里的图片路径和详情文案即可。

## 推荐命名

- 文件夹名使用英文小写和短横线，例如：`hydraulic-deep-drawing-press-500t`
- 图片文件建议使用 `.jpg`、`.webp` 或 `.png`
- 每个产品建议准备：
  - `cover`：产品主图
  - `gallery`：详情页轮播图
  - `details`：图文详情图片、工艺流程图、参数表截图等
  - `description.md`：产品详情文字

## 已建目录

- `content/product-details/`：每个产品的图文详情资料
- `public/images/products/real/`：真实产品图片
- `public/images/factory/real/`：工厂、车间、设备生产现场图片
- `public/images/certificates/`：证书、检测报告、资质图片
- `public/images/clients/`：客户 logo、合作案例 logo
- `public/images/about/`：关于我们页面图片
