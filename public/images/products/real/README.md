# 真实产品图片

建议按产品 ID 建立文件夹，例如：

```text
public/images/products/real/hydraulic-deep-drawing-press-500t/
  cover.jpg
  gallery-01.jpg
  gallery-02.jpg
  gallery-03.jpg
  detail-01.jpg
```

在 `data/products.json` 中引用时，路径写成：

```json
"/images/products/real/hydraulic-deep-drawing-press-500t/cover.jpg"
```
